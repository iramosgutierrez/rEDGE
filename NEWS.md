# rEDGE 0.2.0

* Functions `calculate_EDGE1()` and `calculate_EDGE2()` able to work with 'phylo' and 'multiPhylo' objects
* Wrapper function able to use EDGE1 in phylo and multiphylo; and EDGE2 as phylo, multiPhylo; and adding multiple iterations
* PDi (Phylogenetic Diversity indicator) developed
* Data included: monotremes (Upham et al. 2019; UICN Red List 2025), and cycad data (Forest et al., 2017 and assessments for RLI 2003&2014)
